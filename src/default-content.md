# Welcome to Markdown to PDF

This is a **live preview** markdown editor. Everything you type on the left appears instantly on the right!

## Features

- ✏️ **Live editing** with instant preview
- 🌙 **Dark/Light mode** with auto-detection
- 🖨️ **Print to PDF** with ⌘+P or the button
- 📂 **Load** and 💾 **Save** markdown files

## Text Formatting

You can write **bold**, *italic*, ~~strikethrough~~, and `inline code`.

> This is a blockquote. It's great for highlighting important information.
